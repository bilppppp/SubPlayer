# yt-dlp 可视化界面使用说明

本项目为命令行工具 `yt-dlp` 提供了一个简单易用的 Web 可视化操作界面，让不熟悉命令行的用户也能轻松下载视频和音频。

## 功能特性

-   通过浏览器界面操作，无需记忆复杂命令。
-   支持下载视频。
-   支持提取纯音频并保存为 MP3 格式。
-   实时显示下载进度。
-   自动列出已下载的文件，方便访问。

## 系统要求

-   **Python 3.6+**
-   **yt-dlp**: 必须已在您的系统中安装并能通过命令行直接调用。如果您尚未安装，请参照 [yt-dlp 官方文档](https://github.com/yt-dlp/yt-dlp#installation) 进行安装。

## 安装与启动

1.  **克隆或下载项目**

    将本项目文件放置在您电脑的任意位置。

2.  **安装依赖**

    在项目的根目录 `yt_dlp_webui` 下，我们已经为您创建了一个虚拟环境 `venv` 并安装了必要的 Python 库（Flask, Flask-SocketIO）。

3.  **启动服务**

    打开您的终端（命令行工具），进入 `yt_dlp_webui` 目录，然后执行以下命令来启动 Web 服务：

    ```bash
    # 如果您使用的是 macOS / Linux
    ./venv/bin/python app.py

    # 如果您使用的是 Windows
    .\\venv\\Scripts\\python.exe app.py
    ```

    当您在终端看到类似 `Running on http://127.0.0.1:5000` 的输出时，表示服务已成功启动。

## 如何使用

1.  **打开界面**

    在浏览器中访问 [http://127.0.0.1:5000](http://127.0.0.1:5000)。

2.  **输入链接**

    在 "Enter video URL" 输入框中，粘贴您想要下载的视频链接。

3.  **选择类型**

    在下拉菜单中选择您想要的下载类型：
    -   `Video`: 下载完整视频。
    -   `Audio Only`: 只提取音频，并保存为 MP3 文件。

4.  **开始下载**

    点击 "Download" 按钮。下方的 "Download Progress" 区域将实时显示 `yt-dlp` 的下载日志和进度。

5.  **查看文件**

    下载完成后，新的文件会出现在 "Downloaded Files" 列表中。您可以直接点击文件名来下载或在 `yt_dlp_webui/downloads` 文件夹中找到它。
